# cbp-typography

## Purpose

The Typography component encapsulates predefined styles for different kinds of text that makes up a website, merging design language with CSS and HTML.

## Functional Requirements

* The Typography component contains styles for heading tags and corresponding variants that decouple those styles from the semantic tags.
* The Typography component can also be used to customize other also defines As the highest level container of the application, system-wide functionality may be hoisted to this tag, such as dark mode, performance monitoring, debugging, etc. (TBD)
